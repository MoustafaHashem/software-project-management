\noindent This chapter presents the RACI (Responsible, Accountable, Consulted, Informed) matrix for the Online Course Registration Portal project, clearly defining roles and responsibilities for all project activities and deliverables.

% ============================================================================
\section{RACI Matrix Overview}

\par The RACI matrix is a responsibility assignment matrix that clarifies roles for each task and deliverable:

\begin{description}[leftmargin=2cm, style=nextline]
    \item[R - Responsible] Person(s) who perform the work to complete the task
    \item[A - Accountable] Person who is ultimately answerable for the correct completion (only one A per task)
    \item[C - Consulted] People who provide input and with whom there is two-way communication
    \item[I - Informed] People who are kept up-to-date on progress (one-way communication)
\end{description}

\section{Communication Implications}

\par The RACI matrix informs the communication plan:

\subsection{Consulted (C) Relationships}

\begin{itemize}[leftmargin=*]
    \item Require two-way communication and active collaboration
    \item Input needed before decisions or completion
    \item Regular meetings and reviews scheduled
    \item Timely feedback mechanisms established
\end{itemize}

\subsection{Informed (I) Relationships}

\begin{itemize}[leftmargin=*]
    \item One-way communication (status updates)
    \item Progress reports and notifications
    \item Inclusion in status meetings as observers
    \item Access to project documentation repository
\end{itemize}

% ============================================================================
\section{Conflict Resolution}

\par When responsibilities overlap or conflicts arise:

\begin{enumerate}[leftmargin=*]
    \item The "Accountable" person has final decision authority
    \item Project Manager mediates conflicts between "Responsible" parties
    \item Project Sponsor resolves escalated conflicts
    \item RACI matrix updates require Project Manager and Sponsor approval
\end{enumerate}
